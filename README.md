## 🧪 Research

> **CACHEUK** — *Continual Agentic Contextual Handling and Editing for UK Finance*
>
> Novel framework combining **AlphaEdit + FinDaP + StabilityController** with a custom **Degradation Debt** metric for measuring cumulative edit quality degradation. Fine-tuned on a 2.8B-document UK financial corpus. Benchmarked against ROME, MEMIT, and MEND — outperforms on stability and specificity.
